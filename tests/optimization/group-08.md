# Group 8: Error Handling

### 8.1 Handle 404 gracefully
Try to navigate to a page that does not exist: `http://fixtures/missing-page-abc.html`. Report what happened.

**Verify**: Got an error response without crashing, and the server is still responsive.

### 8.2 Handle missing element gracefully
On any page, try to interact with an element that does not exist on the page. Report what happened.

**Verify**: Got a clear error message rather than a crash or hang.
